## 1. 게임 개요

### 1.1 컨셉

"예쁜 그림책을 펼쳤는데 피가 튀어 있는" 잔혹 동화풍 다크 판타지 뱀서라이크.
영원히 반복되는 저주의 밤에 갇힌 주인공이, 죽을 때마다 기억을 가지고 돌아와
조금씩 강해지며 저주의 비밀을 풀어나가는 게임.

### 1.2 핵심 정보

| 항목 | 내용 |
|------|------|
| 장르 | 뱀서라이크 (Vampire Survivors-like) 로그라이트 |
| 플랫폼 | PC (Steam) |
| 엔진 | Godot 4 (GDScript) |
| 시점 | 탑다운 (Top-down) |
| 아트 스타일 | 픽셀아트 - "피 묻은 동화책" (잔혹 동화풍) |
| 1런 플레이 타임 | 20~30분 |
| 참고 게임 | Vampire Survivors (뱀서라이크 메카닉), 세피리아 (탑다운 픽셀 액션), Cult of the Lamb (잔혹 동화 비주얼), Don't Starve (기묘한 세계관) |
| 개발 범위 | MVP → Steam 얼리 액세스 → 풀 릴리즈 |

### 1.3 핵심 재미 요소

1. **전략적 빌드 구성** - 매 레벨업마다 무기/패시브 선택으로 나만의 빌드 완성
2. **매 런마다 다른 경험** - 절차적 생성 + 랜덤 선택지로 같은 런이 없음
3. **기묘한 세계관 탐험** - 귀엽지만 소름끼치는 세계의 비밀을 매 런마다 조금씩 발견

## 2. 세계관 / 스토리

### 2.1 세계 배경

"에버홀로우(Everhollow)"라는 이름의 작은 마을. 겉보기에는 동화 속처럼 아름다운 마을입니다. 알록달록한 지붕, 꽃이 핀 정원, 친절한 마을 사람들. 하지만 이 마을에는 비밀이 있습니다.

수백 년 전, 마을의 창시자들은 "영원한 풍요"를 위해 고대 마녀 "모르가나"와 계약했습니다. 마녀는 마을에 끝없는 풍요를 주었지만, 대가로 마을 자체를 영원한 저주에 묶어버렸습니다.

매일 밤, 마을은 "진짜 모습"을 드러냅니다. 예쁜 꽃은 이빨 달린 식물로, 귀여운 고양이는 그림자 괴수로, 친절한 이웃은 뒤틀린 괴물로 변합니다. 밤이 끝나면 다시 해질녘으로 돌아갑니다. 아무도 이 사실을 기억하지 못합니다.

아무도... 주인공을 제외하고는.

### 2.2 주인공

저주의 밤에 "깨어난" 자들입니다. 대부분의 마을 사람은 밤이 되면 무의식적으로 괴물로 변하지만, 극소수의 사람들은 저주에 저항하여 인간의 의식을 유지합니다. 이들은 "각성자"라고 불립니다.

각성자들은 밤마다 괴물과 싸워야 하며, 죽으면 다시 해질녘으로 돌아갑니다. 하지만 기억만은 가져갈 수 있습니다. 매 죽음마다 조금씩 더 알게 되고, 조금씩 더 강해집니다.

### 2.3 마녀 모르가나

저주의 근원. 마을 어딘가에 봉인되어 있지만, 그녀의 힘이 매일 밤 마을을 뒤트는 원인입니다. 하지만 진실은 더 복잡합니다 - 모르가나 역시 저주의 피해자일 수 있습니다. 그녀는 마을 사람들의 탐욕에 이용당한 뒤 분노로 저주를 내린 것이며, 저주를 풀려면 단순히 마녀를 처치하는 것이 아니라, 마을의 숨겨진 진실을 밝혀야 합니다.

### 2.4 스토리 진행 구조

```
[1단계] 처음 몇 번의 런
- "여기가 어디지? 왜 괴물이?" → 생존에 집중
- 마을 곳곳에서 일기, 편지, 낙서 등 단서 조각 발견

[2단계] 반복하며 강해진 후
- 마을의 역사 단편 해금 ("모르가나 전설", "창시자의 계약서" 등)
- NPC 대화 해금 (저주에서 깨어난 다른 각성자들)

[3단계] 보스 격파 후
- 각 보스가 죽기 전 남기는 독백으로 진실 조각 공개
- "우리가 먼저 마녀를 배신했다" 등의 반전 암시

[최종] 모르가나 대면
- 마녀를 죽일 것인가, 저주를 다른 방법으로 풀 것인가
- (풀 릴리즈에서 멀티 엔딩 가능)
```

### 2.5 저주의 밤 - 로그라이크 연결

| 게임 메카닉 | 스토리 해석 |
|------------|-----------|
| 런 시작 | 해가 지고 저주의 밤이 시작됨 |
| 적 웨이브 | 밤이 깊어질수록 저주가 강해져 더 강한 괴물 등장 |
| 레벨업 | 각성자의 저주 저항력이 강해지며 새로운 힘 각성 |
| 사망 | 밤이 끝나고 다시 해질녘으로 회귀 |
| 메타 진행 | 이전 밤의 기억이 남아 영구적으로 강해짐 |
| 캐릭터 해금 | 또 다른 각성자를 발견 |
| 보스 처치 | 저주의 핵심 고리 하나를 끊음 |

## 3. 그래픽 스타일 가이드

### 3.1 컨셉: "피 묻은 동화책"

**핵심 원칙:** 첫인상은 귀엽고 예쁘지만, 자세히 보면 소름끼치는 디테일이 숨어 있다.

**귀여움 vs 그로테스크 비율: 7:3**
- 귀여움이 "기본값", 그로테스크가 "발견 요소"
- 6:4 이상이면 "그냥 그로테스크한 게임"으로 인식될 위험
- 8:2 이하이면 차별화가 약해짐

**분위기 레퍼런스 (우선순위):**
1. Cult of the Lamb (귀여운 외형 + 어두운 내용) — 캐릭터 디자인의 기본 방향
2. Don't Starve (기묘한 세계) — 환경/분위기 차용
3. The Binding of Isaac (그로테스크하지만 중독적) — 참고만, 이 수준의 그로테스크는 지양
4. 팀 버튼의 "크리스마스의 악몽" (기괴한 동화) — 색감/무드 참고

**저해상도(320x180) 공포 표현 원칙:**
- 16x16 스프라이트에서 정적 디테일로 공포를 전달하는 것은 불가능
- **디테일이 아닌 행동/색상/타이밍으로 불안감을 전달**한다
  - 애니메이션 기반: idle 시 3~4초마다 눈동자가 따로 움직이는 등 미세 변화
  - 컬러 불일치: 파스텔 톤 형태 + 과채도 핏빛/독색 (귀여운 하트인데 #8B0000)
  - 환경 이벤트: 배경 창문 뒤 그림자 간헐적 움직임 (타일 애니메이션)
- 정적 디테일은 **크게 보이는 곳에 집중**: 64x64 초상화, 기억의 서재 삽화, UI 일러스트

### 3.2 색상 팔레트

#### 기본 배경 톤
```
짙은 보라    #2D1B3D  - 밤하늘, 어두운 영역
어두운 녹색  #1A3A2A  - 풀밭, 숲 (변이된 식물)
자주색      #5C2D6E  - 건물, 구조물의 그림자
짙은 갈색    #3D2415  - 땅, 나무 기둥
중간 보라    #4A3860  - 바닥 타일, 배경과 오브젝트 사이 명도 브릿지
```
> 주의: 배경 톤이 모두 저명도이므로, 중간 보라를 바닥 타일에 활용하여 적/오브젝트와의 명도 분리를 확보한다.

#### 강조색 (이펙트, 오브젝트)
```
밝은 분홍    #FF6B9D  - 저주 마법, 마녀 관련
형광 초록    #39FF14  - 독, 변이, 슬라임
연한 주황    #FF9F43  - 불, 횃불, 폭발
밝은 하늘    #74D0F1  - 성스러운 빛, 회복, 보석
밝은 자홍    #FF6BD6  - 엘리트/보스 오라 (배경 자주색과 톤 충돌 방지를 위해 고명도)
```

#### 캐릭터/오브젝트 톤
```
따뜻한 베이지  #F5DEB3  - 피부, 밝은 오브젝트
파스텔 분홍    #FFB6C1  - 꽃, 귀여운 요소
탁한 빨강     #8B0000  - 피, 위험 표시
밝은 노랑     #FFD700  - 금화, 보물, 크리티컬
순백색        #FFFFFF  - 하이라이트, 눈동자
```

### 3.3 비주얼 가이드라인

#### 캐릭터 디자인
- **2등신 비율** 권장 (32x32에서 3등신은 머리가 너무 작아 특징 전달 불가)
- 큰 눈, 작은 입 (동화적 귀여움)
- 하지만 눈동자에 뭔가 불안한 빛 (동공이 살짝 다르거나, 한쪽 눈이 다른 색)
- 괴물 변이 시: 귀여운 실루엣은 유지하되 이빨, 촉수, 균열 등 그로테스크한 디테일 추가
- 16x16 스프라이트의 디테일은 **1~2개 핵심 포인트에 집중** (예: 이빨꽃의 입 = 3x2px 영역)

#### 환경 디자인
- 낮 (회상/인트로): 파스텔 톤, 꽃과 나비, 동화 마을
- 밤 (인게임): 같은 장소가 뒤틀림. 꽃은 이빨 식물, 나비는 박쥐, 집은 기울어짐
- 배경에 숨겨진 디테일: 벽에 핏자국, 창문 뒤에 비치는 그림자, 웃는 해골 장식

#### 이펙트 디자인
- 채도 높은 밝은 색 사용 (어두운 배경 위에 눈에 확 띄도록)
- 파티클은 둥글고 귀여운 모양 (별, 하트, 물방울) + 비정상적 색 조합
- 예: 적 사망 시 하트 모양 파티클이 터지지만 색은 핏빛 빨강

#### 적 가독성 규칙
- 모든 적 스프라이트는 **1px 밝은 아웃라인을 에셋 자체에 내장**한다 (셰이더가 아닌 스프라이트 단계 처리, MultiMesh 호환)
- 어두운 적(그림자 고양이 등)은 **눈이나 특정 부위에 강조색 필수** (예: 형광 초록 눈)
- 엘리트 몬스터는 아웃라인 + 밝은 자홍(#FF6BD6) 오라로 이중 차별화
- 보스는 크기(64x64, 96x96) 자체가 차별화 요소이므로 아웃라인 불필요, 전용 이펙트로 구분

#### 폰트/UI 디자인
- **2중 폰트 전략** (한글 가독성 확보 필수):
  - 손글씨 느낌 픽셀 폰트: 게임 로고, 화면 제목, 캐릭터/무기 이름에만 사용
  - 가독성 높은 깔끔한 픽셀 폰트: 효과 설명, 수치, 시스템 텍스트 등 본문에 사용
- UI 프레임: 낡은 동화책 장식, 넝쿨과 가시 모티프
- 버튼: 둥근 모양 + 약간 삐뚤어진 테두리 (완벽하지 않은 느낌)

## 4. 핵심 게임 루프

### 4.1 런(Run) 구조

```
[해질녘] → 밤 시작 카운트다운 (3초)
  ↓
[밤 시작] → 자동 공격 시작, 적 스폰 시작
  ↓
[웨이브 진행] ← ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐
  │                                    │
  ├→ 적 처치 → 경험치 보석 드롭         │
  ├→ 보석 수집 (마그넷)                  │
  ├→ 경험치 가득 → [레벨업 선택]         │
  │   └→ 무기/패시브 선택 → 복귀 ─ ─ ─ ┘
  │
  ├→ 5분마다: 엘리트 웨이브
  ├→ 보물 상자 스폰 (엘리트 처치/시간)
  │   └→ 획득 시 강력한 보상 선택
  │
  ├→ 10분: 중간 보스 등장 (MVP 1종)
  │   └→ 처치 시 확정 보물 상자 + 스토리 단서
  │
  └→ 20분: 최종 보스 등장
      ├→ 처치 → [클리어 결과]
      └→ 사망 → [사망 결과]

[결과 화면] → 기억 포인트 정산 → [메타 진행] → [해질녘으로 복귀]
```

### 4.2 자동 공격 시스템

- 플레이어 조작: **이동만** (WASD / 방향키 / 좌스틱)
- 무기는 자동 발동 (쿨다운 기반)
- 무기마다 고유한 발동 패턴:
  - 가장 가까운 적 방향으로 발사 (단검)
  - 플레이어 주변 회전 (성경, 마늘 오라)
  - 랜덤 위치에 투하 (성수)
  - 플레이어 이동 방향으로 발사 (채찍)
- 최대 장착 무기: 6종
- 최대 패시브 아이템: 6종

### 4.3 웨이브 시스템

| 시간 | 이벤트 | 적 밀도 |
|------|--------|--------|
| 0:00~2:00 | 약한 적 소수 등장, 적응 구간 | 낮음 |
| 2:00~5:00 | 적 종류 증가, 수량 증가 | 중간 |
| 5:00 | **엘리트 웨이브 1** - 강화 몬스터 등장 | 높음 |
| 5:00~10:00 | 새로운 적 유형 추가, 밀도 증가 | 높음 |
| 10:00 | **중간 보스 등장** (MVP: 마을 영주) | 최고 |
| 10:00~15:00 | 보스 처치 후 잠시 소강 → 다시 증가 | 높음 |
| 15:00 | **엘리트 웨이브 2** | 매우 높음 |
| 15:00~20:00 | 최고 난이도, 화면 가득 적 | 극한 |
| 20:00 | **최종 보스 등장** (MVP: 마녀의 사자) | 극한 |
| 20:00~ | 보스 미처치 시 무한 웨이브 (생존 모드) | 극한 |

### 4.4 경험치 & 레벨업

**경험치 보석:**
- 파란 보석 (소): 1 XP - 일반 몬스터 드롭
- 초록 보석 (중): 5 XP - 다수 몬스터, 파괴 오브젝트
- 빨간 보석 (대): 20 XP - 엘리트, 보스

**마그넷 시스템:**
- 기본 흡수 범위: 32px 반경
- 보석이 범위 내 진입 시: 느리게 시작 → 가속 → 플레이어에 도달
- 패시브 "자석 범위 증가": Lv당 +30% (최대 Lv5, +150%)
- 마그넷 필드 아이템: 획득 시 화면 전체 보석 즉시 흡수

**레벨업 선택:**
- 게임 일시정지
- 카드 **3장 고정** 표시 (320px 폭 제약: 카드 80px x 3 + 간격 = 280px 이내)
- 선택지: 새 무기 / 기존 무기 레벨업 / 새 패시브 / 기존 패시브 레벨업
- 무기 최대 레벨: Lv8
- 패시브 최대 레벨: Lv5

### 4.5 데미지 숫자 표시

- 적중 시 대상 머리 위에 숫자 팝업
- **폰트**: 가독성 폰트 계열의 BMFont 변형 (8x10px/digit). 각 숫자의 baseline을 1px씩 불규칙하게 배치하여 동화책 느낌 부여. 1px 드롭 셰도우 내장.
- **Tween 연출**: 팝업 시 position에 최대 1px 랜덤 오프셋, 크리티컬은 scale에 미세 불균일(±0.1) 추가
- 일반 데미지: 흰색, 위로 떠오름 (0.3초) → 페이드아웃 (0.2초)
- 크리티컬: 금색, 1.5배 크기, 흔들림 → 떠오름 → 페이드아웃
- 회복: 녹색, 아래에서 위로 부드럽게 상승
- 동시 표시 최대 20개, 위치 오프셋으로 겹침 방지

### 4.6 화면 밖 적 인디케이터

- 일반몹 밀집 (10마리+): 화면 가장자리에 빨간 삼각형
- 엘리트: 보라색 삼각형 + 펄스 (항상 표시)
- 보스: 금색 해골 아이콘 + 펄스 (항상 표시)
- 거리에 따라 불투명도 변화 (가까울수록 선명)
- 설정에서 ON/OFF 가능

## 10. UX 시나리오

### 10.1 화면 흐름

```
[Godot 스플래시] → [팀 로고] → [타이틀 화면]
                                     │
                                     ├→ [시작] → [캐릭터 선택] → [스테이지 선택] → [인게임]
                                     ├→ [기억의 제단] (영구 업그레이드)
                                     ├→ [기억의 서재] (스토리 열람)
                                     ├→ [설정]
                                     └→ [종료]

[인게임] 루프:
  적 처치 → 보석 드롭 → 마그넷 흡수 → 경험치 → [레벨업 선택] → 복귀
  엘리트/시간 → 보물 상자 스폰 → 접촉 → [보물 상자 보상] → 복귀
  10분 → [보스 경고] → 보스전 → 처치 → 보물 상자 → 복귀
  20분 → [최종 보스] → 처치 → [클리어 결과] → [메타 정산] → [타이틀]
  사망 → [사망 결과] → [메타 정산] → [타이틀]
  ESC → [일시정지] → 계속/설정/포기
```

### 10.2 화면별 상세

#### S1. 타이틀 화면
- 배경: 에버홀로우 마을의 낮 모습 (아름다운 동화풍), 시간이 지나면 서서히 밤으로 전환되며 뒤틀림
- 게임 로고: "저주받은 밤" (손글씨 픽셀 폰트, 넝쿨 장식)
- BGM: 잔잔한 오르골 → 점점 불협화음으로 변함
- 메뉴 버튼: [시작] [기억의 제단] [기억의 서재] [설정] [종료]

#### S2. 캐릭터 선택
- 해금된 캐릭터 나열 (미해금은 실루엣 + 해금 조건 표시)
- 캐릭터 선택 시: 초상화(64x64) + 이름 + 초기 무기 + 고유 패시브 + 스탯 막대 표시
- 캐릭터 옆에 짧은 대사 ("이번 밤은... 끝내겠어")
- [선택] → 스테이지 선택으로 이동

#### S3. 스테이지 선택
- 에버홀로우 마을 지도 형태 (동화책 지도 스타일)
- 해금 구역 클릭 가능, 미해금 구역은 안개/자물쇠
- 각 구역: 이름 + 난이도 표시 + 짧은 설명
- [출발] → 인게임 시작

#### S4. 인게임 (핵심)
**HUD 배치 (화면 점유율 20% 이하 목표):**
```
[HP바 ■■■■■□□□□□] [무기아이콘x6] [⏱ 00:00] [💀 0 🪙 0]
[XP ════════════ Lv.1]

              (게임 플레이 영역)
         (플레이어 중앙, 적 사방에서 접근)
         (경험치 보석 흩어짐)
         (데미지 숫자 팝업)
         (화면 가장자리: 적 인디케이터)

[패시브아이콘x6 (소형)]
```
> 변경사항: 무기 슬롯을 상단 HP바 옆으로 통합 (소형 아이콘), 미니맵 제거 (화면 밖 적 인디케이터로 대체), 패시브 슬롯 소형화. 상단 ~10% + 하단 ~6% = 약 16%.

**인게임 연출:**
- 게임 시작 시 "해가 지는" 짧은 연출 (1초)
- 첫 적 등장까지 5초 유예 (이동 적응)
- 적 처치 시: 귀여운 파티클 터짐 (하트, 별) + 비정상적 색 (핏빛)
- 대량 처치 시: 화면 흔들림 + 쾌감 이펙트

#### S5. 레벨업 선택
- 게임 일시정지 + 배경 어두워짐
- 중앙에 카드 3장 고정 (동화책 카드 디자인, 넝쿨 테두리)
- 각 카드: [아이콘] + [이름] + [레벨 표시] + [효과 텍스트 1줄]
- 카드 호버: 살짝 확대 + 상세 설명 팝업
- 카드 클릭: 선택 이펙트 → 게임 재개
- 신규 무기: 카드 테두리 금색 반짝임
- 레벨업: 카드에 현재 레벨 → 다음 레벨 표시

#### S6. 보물 상자 보상
- 게임 일시정지
- 화면 중앙: 상자 열림 애니메이션 (6프레임)
- 빛 파티클 + 보상 카드 2~3장 표시
- 일반 레벨업보다 강력한 선택지 (고레벨 무기, 희귀 패시브, 대량 회복 등)
- 선택 후 게임 재개

#### S7. 보스 등장 연출
- "WARNING!" 텍스트 (빨강, 흔들림) + 화면 가장자리 빨간 점멸
- 1.5초간 표시 후 보스 등장
- 보스 이름 + HP바 화면 하단에 표시 (이름 양옆에 장식)
- 보스 전용 BGM 전환

#### S8. 사망 / 런 종료

**사망 시:**
- 슬로우 모션 (0.5초) → 사망 애니메이션 → 화면 페이드
- "밤이 끝났다..." 텍스트 표시

**결과 화면:**
```
┌─────────────────────────────┐
│      이번 밤의 기억            │
│                              │
│  ⏱ 생존 시간: 12:34          │
│  💀 처치 수: 847             │
│  🪙 획득 기억 조각: 156       │
│                              │
│  사용한 무기:                  │
│  [가위 Lv5] [양초 Lv3] [꽃다발 Lv2] │
│                              │
│  발견한 단서:                  │
│  📜 "마을 사람의 일기 #3"      │
│                              │
│        [계속]                  │
└─────────────────────────────┘
```

**클리어 시:**
- 보스 사망 연출 → "저주가 약해지고 있다..." → 결과 화면
- 결과 화면에 "클리어!" 표시 + 보너스 기억 조각

#### S9. 기억의 제단 (메타 진행)
- 제단 배경 (양초로 둘러싸인 석판)
- 업그레이드 목록 좌측 나열
- 각 업그레이드: [아이콘] [이름] [현재 Lv/최대 Lv] [효과] [비용] [구매 버튼]
- 보유 기억 조각 상단 표시
- 구매 시: 빛 이펙트 + 레벨 업 연출

#### S10. 기억의 서재 (스토리)
- 동화책 형태의 UI
- 발견한 단서를 페이지 넘기듯 열람
- 미발견 페이지는 잠금 표시 (흐린 실루엣)
- 각 단서: 삽화 + 텍스트 (동화책 서술체)

#### S11. 설정
- BGM 볼륨 / SFX 볼륨 (슬라이더)
- 화면 모드 (풀스크린 / 창모드)
- 해상도 선택
- 적 인디케이터 ON/OFF
- 화면 흔들림 ON/OFF
- 데미지 숫자 ON/OFF
- 조작 설정 (키 리바인딩)
- 언어 (한국어/영어)

#### S12. 일시정지
- ESC로 토글
- 반투명 검정 오버레이
- 중앙 메뉴: [계속] [설정] [포기 (메인 메뉴로)]
- 현재 런 요약: 무기 목록, 패시브 목록, 생존 시간, 레벨

### 해상도 및 스케일링 상세

| 설정 | 값 |
|------|-----|
| stretch/mode | canvas_items |
| stretch/aspect | keep |
| stretch/scale_mode | integer |
| snap_2d_transforms_to_pixel | true |
| snap_2d_vertices_to_pixel | true |
| default_clear_color | #1A0F28 (극짙은 보라, 레터박스 영역) |

**레터박스 규칙:**
- 1080p(x6), 1440p(x8)에서는 레터박스 없이 딱 맞음
- 비표준 해상도(예: 1366x768)에서는 정수 배율 적용 후 남는 영역에 #1A0F28 표시
- 풀 릴리즈에서 SubViewport 구조로 전환하여 동화책 테두리 장식 추가 검토

### 환경 연출 구조

- 환경 연출은 **별도 매니저를 두지 않고 스테이지 씬 내부에서 처리**한다
- `StageEnvironmentData` Resource(.tres)에 환경 설정값을 분리 (CanvasModulate 색상, 조명, 안개 등)
- 각 스테이지 씬은 `StageEnvironment` 노드를 포함하여 Resource를 읽고 환경을 자동 설정
- 새 스테이지 추가 시 .tres 파일만 추가하면 됨 (OCP 준수, 매니저 신설 불필요)
